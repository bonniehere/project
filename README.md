<h1>병원 실시간 진료 예약 구현 서비스</h1>


### 🏣 개요 
병원 실시간 진료 예약 구현 서비스를 제공하기 위해 중앙대학교 광명병원 사이트를 카피




### 🔧 개발 환경
- eclipes 
- JAVA
- Spring frame work (MVC)
- MySQL




### 🗓 프로젝트 일정
2023/12/12 ~ 2023/12/27




### 👩‍💻 팀원
정순욱(팀장) 강민채
